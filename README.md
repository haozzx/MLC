    如何进行挖矿?
    方法一：利用钱包挖矿
	下载MLC.rar
    解压并运行milancoin-qt.exe
    打开调试窗口
    （帮助 -> 调试 ->控制台）
    在提示符">"后输入命令：setgenerate true -1

    查看如下内容
